## 🔬 影响力论文

### 1. Surface code logical operations on a superconducting quantum processor

**来源**: arXiv quant-ph  
**时间**: 刚刚  
**链接**: https://arxiv.org/abs/2607.01473  

在107量子比特超导量子处理器上实验实现了基于表面码的逻辑操作，通过码变形和晶格手术展示了容错量子计算的关键模块，所有逻辑操作均在保持纠错保护下完成。

### 2. LUCI on IBM Hardware: Error Suppression with Almost Half Syndrome Density

**来源**: arXiv quant-ph  
**时间**: 刚刚  
**链接**: https://arxiv.org/abs/2607.01887  

LUCI框架在IBM硬件上实现纠错，将症状密度降低近一半，维持时空距离并提升对硬件缺陷的鲁棒性。

### 3. Design rules for fault-tolerant multi-gate teleportation

**来源**: arXiv quant-ph  
**时间**: 刚刚  
**链接**: https://arxiv.org/abs/2607.01342  

多门隐形传态将n个远程门压缩到单对纠缠比特中，节省n-1对纠缠资源，但引入权重n的关联错误。研究给出容错最大包尺寸设计规则：对于旋转表面码，$\nmax^{\text{corr}}(d) = \lceil d/2 \rceil$。

### 4. High-Precision Calibration Workflow Achieves Above $99.9\%$ CZ Gate Fidelity on a Scalable Superconducting Processor

**来源**: arXiv quant-ph  
**时间**: 刚刚  
**链接**: https://arxiv.org/abs/2607.01422  

闭环工作流程将可扩展超导处理器上 CZ门的保真度提升至99.9%以上，通过回波技术抑制退相干误差，实现高精度通用两比特门校准。

### 5. Automated logical Clifford gadgets for heterogeneous architectures via chain maps

**来源**: arXiv quant-ph  
**时间**: 刚刚  
**链接**: https://arxiv.org/abs/2607.02482  

提出一种自动化框架，通过链映射合成任意CSS码间的逻辑CNOT电路，适用于异构量子架构，无需依赖结构相近码族，显著扩展了跨码纠缠操作的可能性。
